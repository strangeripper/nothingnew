Decision Support Systems (DSS)

A Decision Support System is an interactive tool that helps people make better choices using data, analytical models, and intuitive interfaces.

Core components

  • Data: internal + external sources.
  • Models: statistics, optimization, simulation.
  • Interface: dashboards, reports, visualizations.

Common types

  • Data-driven • Model-driven • Knowledge-driven
  • Communication-driven • Document-driven
